## BigInteger Calculator (C++)
## Overview

This project is a menu-driven Big Integer Calculator built in C++.
It performs arithmetic operations on numbers larger than built-in data types like long long, enabling high-precision calculations.

It uses a custom BigInteger class that supports operator overloading for natural expression handling.

## Features
- ➕ Addition of large integers
- ➖ Subtraction of large integers
- ✖️ Multiplication
- ➗ Division (with zero-check protection)
- 📊 Modulus operation
- ⚖️ Number comparison (>, <, ==)
- ⬆️ Increment operator (++a)
- ⬇️ Decrement operator (--a)
- ➖ Unary minus support (-a)
- 🧾 Interactive menu-driven interface

## Key Concepts Used
- Object-Oriented Programming (OOP)
- Operator Overloading (+, -, *, /, %, ++, --, ==, >, -)
- Custom class implementation (BigInteger)
- String-based number storage for arbitrary precision
- Input validation (e.g., division by zero handling)
- Infinite loop menu system with exit control

 ## Why this project is useful

- Normal C++ data types like int and long long have size limits.
- This project solves that problem by allowing:

- Very large number computations (hundreds or thousands of digits)
- Real-world simulation of how calculators and crypto systems handle big numbers
- Strong understanding of memory handling and class design
